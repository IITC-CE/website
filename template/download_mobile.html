IITC Mobile

IITC Mobile works as a simple web browser, with the IITC browser add-on embedded within it.

Android

Requires Android 4.2.2 (Jelly Bean) at a minimum.

Note: Different sources use different signature keys:

  • Google Play
  • official F-Droid repo
  • IITC F-Droid repo and our .apk

So installing the update over the application from another source is not possible. You must first uninstall the previous version.
Use plugin for backup/restore to migrate your data.

Get it on official F-Droid repo or IITC F-Droid repo

F-Droid is an installable catalogue of FOSS (Free and Open Source Software) applications for the Android platform. The client makes it easy to browse, install, and keep track of updates on your device.

Unlike the official F-Droid repository, the IITC F-Droid repository also provides beta versions of the mobile app and allows you to receive updates instantly.

iOS

Requires iOS 9.0 or later. Compatible with iPhone, iPad, and iPod touch.

Get it on Apple Store

Testing IITC Mobile with TestFlight

Since v3.3.1, Apple has rejected new IITC Mobile versions for not providing Apple SignIn. To use/test new version, please join testflight.
