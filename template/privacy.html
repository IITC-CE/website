Privacy Policy for IITC-CE Applications

Effective Date: 29 August 2025
Last Updated: 30 March 2026

Overview

This Privacy Policy applies to all IITC-CE applications, including IITC Mobile and IITC Prime.

IITC-CE ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we handle information in connection with our applications.

Data Collection by IITC-CE Applications

We do not collect, store, or transmit any personal data or user information.

IITC-CE applications function as enhanced web browsers that:

  • Load intel.ingress.com (Niantic's official Ingress Intel website) in a webview
  • Apply userscripts to enhance the interface with additional features
  • Do not collect any personal information from users
  • Do not store user data on our servers
  • Do not track user behavior or usage patterns
  • Do not require user accounts or registration

Crash and Error Reporting (Sentry)

To improve app stability and diagnose issues, IITC-CE applications may use Sentry, a third-party error tracking service, to automatically send crash and error reports when the application encounters a problem.

Each error report may include the following non-personal technical data:

  • Error details: error type, stack trace, and related diagnostic information
  • App information: application version, build number, and foreground/background state
  • Device information: device manufacturer, model, screen resolution, processor count, total and available memory and storage
  • Operating system: OS name, version, build number, and kernel version
  • Runtime context: battery level, charging status, network connection type (e.g. Wi-Fi or cellular), device orientation, and device locale/timezone
  • Random device identifier: a randomly generated ID used solely to group errors from the same device — it is not linked to any user account or personal identity

What is NOT collected:

  • No personally identifiable information (names, email addresses, or user accounts)
  • No IP addresses — IP address storage is disabled in our Sentry configuration
  • No user-generated content or gameplay data
  • No screenshots, screen recordings, or session replays — these Sentry features are explicitly disabled

This data is used solely to identify and fix application errors and is not used for analytics, advertising, or profiling. Data is processed by Sentry in accordance with their Privacy Policy.

How IITC-CE Applications Work

IITC-CE applications are specialized browsers that:

  • Display intel.ingress.com with enhanced userscripts running in the webpage context
  • May download userscript updates from our server at iitc.app (which only serves static files and does not log users)
  • May cache web resources temporarily on your device for performance
  • May request standard device permissions for functionality (data remains on your device)
Important: The userscripts work within the webpage like any JavaScript code - they do not intercept network traffic or modify data transmission.

Third-Party Services - Ingress Intel

When you use IITC-CE applications, you are directly accessing intel.ingress.com, operated by Niantic, Inc.:

  • All user data collection and storage is handled by intel.ingress.com according to Niantic's Privacy Policy
  • We have no access to or control over the data processed by intel.ingress.com
  • You interact directly with Niantic's service through our enhanced interface
Important: Please review Niantic's Privacy Policy to understand what data they collect when you use intel.ingress.com.

Children's Privacy

IITC-CE applications do not collect any information from users of any age. However, intel.ingress.com (accessed through our applications) is operated by Niantic and may have age restrictions. Parents should review Niantic's terms and privacy policy.
